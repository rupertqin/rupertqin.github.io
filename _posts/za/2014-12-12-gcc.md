[Install GCC from source code](https://gcc.gnu.org/wiki/InstallingGCC)

    tar xzf gcc-4.6.2.tar.gz
    cd gcc-4.6.2
    ./contrib/download_prerequisites
    cd ..
    mkdir objdir
    cd objdir
    $PWD/../gcc-4.6.2/configure --prefix=$HOME/gcc-4.6.2
    make
    make install
