[Tech Tip: Really Simple HTTP Server with Python](http://www.linuxjournal.com/content/tech-tip-really-simple-http-server-python)

`$ cd /home/somedir`

`$ python -m SimpleHTTPServer`

default port is "8000"

or customize a port:

`$ python -m SimpleHTTPServer 8080`
